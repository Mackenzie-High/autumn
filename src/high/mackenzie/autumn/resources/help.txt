How to Use:

    To create a project folder in the current directory:

        autumn create <name>

    To run a project:

        autumn run <argument-0> <argument-1> ... <argument-n>

        Note: The current directory must be a project folder.

    To test a project:

        autumn test

        Note: The current directory must be a project folder.

    To compile a project to a JAR file:

        autumn compile

        Note: The current directory must be a project folder.

    To execute a JAR file containing a previously compiled program:

        autumn <path-0> <path-1> ... <path-n> execute <argument-0> <argument-1> ... <argument-n>

        Each <path> must be the path to a JAR file.

        Multiple paths are allowed, since some of them may be libraries.

        Libraries may be created using Autumn or other JVM languages.

    To print the version:

        autumn version

    To print this help message:

        autumn help

    To print the full license:

        autumn license
