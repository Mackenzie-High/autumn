How to Use:

    To create a project folder in the current directory:

        autumn create <name>

    To run a project:

        autumn run <argument-0> <argument-1> ... <argument-n>

        Note: The current directory must be a project folder.

    To test a project:

        autumn test

        Note: The current directory must be a project folder.

    To print the version:

        autumn version

    To print this help message:

        autumn help

    To print the full license:

        autumn license


Have a great day and remember that Jesus loves you!
