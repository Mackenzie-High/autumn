# Sequence Statement

## Summary

A sequence-statement executes a series of statements one after another.

## Syntax

```plain
{
    <i>[statement](TextPage.html?page=Statement)<sub>1</sub></i>
    <i>[statement](TextPage.html?page=Statement)<sub>2</sub></i>
    <i>[statement](TextPage.html?page=Statement)<sub>n</sub></i>
}
```

## AST Class

autumn.lang.compiler.ast.nodes.SequenceStatement

## Details
+ There can be zero or more statements in the sequence.
