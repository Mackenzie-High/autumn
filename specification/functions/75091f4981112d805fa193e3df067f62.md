## Summary

This function retrieves the value of an array element.

## Signature

get (array : [BigInteger](https://docs.oracle.com/javase/7/docs/api/java/math/BigInteger.html)[], index : int) : [BigInteger](https://docs.oracle.com/javase/7/docs/api/java/math/BigInteger.html)

## Formals

is the array that contains the element.
is the location of the element in the array.

## Returns

Return the value stored in the <i>array</i> element that is identified by the given <i>index</i>.
