## Summary

This function determines whether a [Class](https://docs.oracle.com/javase/7/docs/api/java/lang/Class.html) represents the type of a design.

## Signature

isDesignType (type : [Class](https://docs.oracle.com/javase/7/docs/api/java/lang/Class.html)) : boolean

## Formals

Parametertypemay be a design-type.

## Returns

Return true, iff <i>type</i> is the type of a design that was created using Autumn.

## Throws

[NullPointerException](https://docs.oracle.com/javase/7/docs/api/java/lang/NullPointerException.html)if <i>type</i> is null.
