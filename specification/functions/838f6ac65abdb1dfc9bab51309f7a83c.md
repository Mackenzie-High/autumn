## Summary

(Under Development) - This function replaces escape sequences with their expansions.

## Signature

escape (string : [String](https://docs.oracle.com/javase/7/docs/api/java/lang/String.html)) : [String](https://docs.oracle.com/javase/7/docs/api/java/lang/String.html)

## Formals

+ Parameter <i>string</i> is the string that contains the escape sequences.

## Returns

Return the <i>string</i> with the escape sequences therein replaced with their expansions.

## Throws

+ [NullPointerException](https://docs.oracle.com/javase/7/docs/api/java/lang/NullPointerException.html), if <i>string</i> is null.
