## Summary

This function determines whether a [Class](https://docs.oracle.com/javase/7/docs/api/java/lang/Class.html) represents the type of a record.

## Signature

isRecordType (type : [Class](https://docs.oracle.com/javase/7/docs/api/java/lang/Class.html)) : boolean

## Formals

may be a record-type.

## Returns

Return true, iff <i>type</i> is a record-type (tuple | struct | design).
