## Summary

This function sets the value of an array element.

## Signature

set (array : [Object](https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html)[], index : int, value : [Object](https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html)) : [Object](https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html)[]

## Formals

is the array that contains the element to set.
is the location of the element in the array.
is the value to replace the element with.

## Returns

The modified array is returned.
