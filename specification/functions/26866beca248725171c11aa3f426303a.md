## Summary

This function determines whether a [Class](https://docs.oracle.com/javase/7/docs/api/java/lang/Class.html) represents the type of a struct.

## Signature

isStructType (type : [Class](https://docs.oracle.com/javase/7/docs/api/java/lang/Class.html)) : boolean

## Formals

+ Parameter typemay be a struct-type.

## Returns

Return true, iff <i>type</i> is the type of a struct that was created using Autumn.

## Throws

+ [NullPointerException](https://docs.oracle.com/javase/7/docs/api/java/lang/NullPointerException.html): if <i>type</i> is null.
